# Piyal Gupta — Advisory

A single-page, neumorphic consulting site for **Piyal Gupta** (Co-Founder &amp; CEO, Kneuralabs),
positioning his enterprise + AI-governance advisory services.

## What's here

- `index.html` — self-contained landing page (no build step, no dependencies).
  Open it directly or serve it via GitHub Pages.

## Design

- **Style:** Neumorphism (soft UI) — a single stone-toned canvas with paired
  light/dark shadows for extruded and inset surfaces.
- **Palette &amp; type:** reuses the CV's tokens (stone / ink / forest-green accent,
  Inter + JetBrains Mono) so the page reads as a sibling of
  [the CV](https://piyalgupta.github.io/cv/).
- **Dark mode:** toggle in the header, persisted to `localStorage`.
- **Motion:** scroll-reveal with `prefers-reduced-motion` respected.
